# 📚 출석 및 학습 관리 시스템

학생의 출석을 관리하고 학습 현황을 추적하는 완벽한 웹 시스템입니다.

## ✨ 주요 기능

### 👨‍🎓 학생 기능
- **회원가입 및 로그인**: 이메일과 비밀번호를 사용한 사용자 인증
- **강의 등록**: 학생이 수강할 강의 선택 및 등록
- **출석 확인**: 강의실 환경 확인 + 얼굴 인식 + 코드 검증을 통한 안전한 출석
- **퇴장**: 강의 종료 5분 전부터 코드 입력으로 안전한 퇴장
- **공가 신청**: 경조사, 병원, 자격증-시험 등으로 분류된 공가 신청
- **통계 조회**: 출석률, 지각, 결석, 공가 현황 확인

### 👨‍💼 관리자 기능
- **강의 관리**: 강의 추가 및 삭제
- **학생 출석 현황**: 학생별, 강의별 출석 조회
- **통계 대시보드**: 전체 학생 수, 평균 출석률 등 통계

## 🚀 빠른 시작

\`\`\`bash
npm install
npm run dev
\`\`\`
