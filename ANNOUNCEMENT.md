# 📢 출석 및 학습 관리 시스템 - 공개 공지

## 🎉 시스템 완성!

저희는 **학생 출석 및 학습 관리 시스템**의 완전한 버전을 완성하여 공개합니다!

---

## ✨ 주요 기능

### 👨‍🎓 학생용 기능
- ✅ **회원가입 & 로그인** - 이메일 기반 사용자 인증
- ✅ **강의 등록** - 학생이 원하는 강의를 자유롭게 선택 등록
- ✅ **출석 확인** - 한 번의 클릭으로 출석 기록 (시간 자동 저장)
- ✅ **통계 조회** - 수강 강의, 출석 횟수, 출석률 실시간 확인
- ✅ **강의 중단** - 필요시 강의 중단 신청

### 👨‍💼 관리자용 기능
- ✅ **강의 관리** - 강의 추가 및 삭제
- ✅ **학생 관리** - 등록된 학생 현황 파악
- ✅ **출석 현황 조회** - 학생별, 강의별 상세 출석 현황
- ✅ **통계 대시보드** - 전체 학생 수, 평균 출석률 등 관리 통계

## 📚 사용 가이드

### 학생이 할 일
1. **로그인** - 계정으로 로그인
2. **강의 등록** - "+ 강의 등록" 버튼으로 원하는 강의 선택
3. **출석 확인** - 강의 클릭 → "✅ 출석 확인하기" 버튼 클릭
4. **통계 확인** - 대시보드에서 수강 강의, 출석 현황 실시간 조회

### 관리자가 할 일
1. **로그인** - 관리자 계정으로 로그인
2. **강의 관리** - 강의 추가/삭제
3. **학생 관리** - 학생 출석 현황 조회
4. **통계 확인** - 전체 통계 및 학생별 상세 현황

## 🛠 기술 사양

| 항목 | 정보 |
|------|------|
| **프론트엔드** | Next.js 16, React 19, TypeScript |
| **스타일링** | Tailwind CSS |
| **상태관리** | Zustand |
| **UI 라이브러리** | Lucide React Icons, React Hot Toast |
| **데이터 저장** | Browser localStorage (향후 Supabase 연동 예정) |
| **배포 플랫폼** | Vercel |

## 📋 구현된 페이지

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 홈 | `/` | 로그인/회원가입 선택 |
| 로그인 | `/login` | 사용자 로그인 |
| 회원가입 | `/signup` | 새 계정 생성 |
| 학생 대시보드 | `/student` | 강의 목록, 통계, 강의 등록 |
| 강의 상세 | `/student/course/[id]` | 출석 확인, 강의 정보, 출석 현황 |
| 관리자 대시보드 | `/admin` | 강의 관리, 학생 관리, 통계 |
